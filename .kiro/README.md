# .kiro Directory

This directory contains Kiro-specific configuration and steering files for the Multilingual Mandi project.

## Structure

```
.kiro/
├── README.md           # This file
├── steering/           # Steering files for AI guidance
│   ├── ai-pricing.md   # AI pricing guidelines
│   ├── translation.md  # Translation best practices
│   └── negotiation.md  # Negotiation strategies
└── settings/           # Kiro settings and configurations
    └── project.json    # Project-specific settings
```

## Steering Files

Steering files provide context and guidelines for AI agents working on this project. They help maintain consistency and quality across different AI-powered features.

### ai-pricing.md
Guidelines for AI-powered price generation, including:
- Market analysis principles
- Seasonal factor considerations
- Regional pricing variations
- Quality and freshness indicators

### translation.md
Best practices for multilingual translation, including:
- Cultural adaptation strategies
- Indirect speech patterns
- Market-appropriate terminology
- Context preservation techniques

### negotiation.md
Negotiation strategies and patterns, including:
- Multi-round negotiation flows
- Floor price protection
- Polite counter-offer generation
- Cultural sensitivity in bargaining

## Settings

Project-specific configurations for Kiro integration and AI behavior customization.

## Usage

These files are automatically loaded by Kiro when working on the project, providing context-aware assistance for:
- Code generation and modification
- AI service optimization
- Cultural and linguistic accuracy
- Market-specific feature development

## Maintenance

Update steering files when:
- Adding new AI features
- Expanding to new languages or regions
- Modifying negotiation strategies
- Updating market analysis criteria
